This repository contains the Python code for Multi-chat room.

Follow the below steps for executing the code:
1. First open terminal and go to path directory where the server client files are existing
2. Enter Server.py and then your Chat Server will be running
3. In another terminal go to the path directory where Server and Client files are existing
4. Enter Client.py and a Chat UI will pop up where you can chat with other clients. 

You can create multiple clients by opening multiple command terminals. 
You can view the connected clients from the Server side as well

To exit Chat UI , you can press the quit button on the UI pop up
